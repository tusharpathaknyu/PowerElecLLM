# Contributing to PowerElecLLM

Thank you for your interest in contributing to PowerElecLLM!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/PowerElecLLM.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit: `git commit -m "Add your feature"`
6. Push: `git push origin feature/your-feature-name`
7. Open a Pull Request

## Development Setup

```bash
conda env create -f environment.yml
conda activate power_electronics
```

## Code Style

- Follow PEP 8 for Python code
- Use meaningful variable names
- Add docstrings to functions
- Include comments for complex logic

## Testing

- Test your changes with example circuits
- Ensure generated code runs in PySpice
- Validate power analysis results

## Pull Request Process

1. Update README.md if needed
2. Add tests if applicable
3. Ensure all checks pass
4. Request review

Thank you for contributing!
